# Supporting Artifacts

This folder contains two small supplementary artifacts that strengthen the paper
without widening the repo beyond the publication bundle.

- `repeatability_summary_v1.json`
  Three-seed repeatability summary for selected models in the threshold bundle.
- `qwen3_8b_role_aware_comparison.json`
  The weight-quantization comparison cited in the paper’s independent
  confirmation section.

The paper bundle remains the primary source of truth. These files are secondary.
